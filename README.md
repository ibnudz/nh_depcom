## How to Run?
    # use python 3 version
    git clone https://github.com/ibnudz/nh_depcom.git
    cd nh_depcom
    pip install -r requirements.txt
    # fix pipe for deploy
    python get-deploy.py
    # get combo ninja and calculate
    python get-combo.py
